# checker-board
Deployed: https://checkers-dusky.now.sh/

## Installation

```bash
$ https://github.com/cjbt/checker-board.git

// start server
$ cd checker-board
$ yarn install
$ yarn server

// start client
$ cd client
$ yarn install
$ yarn start
```

### Env vars
```
REACT_APP_BASE_URL=http://localhost:4000/api/state
```

## Testing

```bash
$ yarn test
```
